## 📖 About The Project

**Point5 Media Productions** is a multidisciplinary creative consultancy and results-driven digital marketing agency. We assist brands in growing, connecting, and thriving in the digital world. 

This repository contains the official frontend website for Point5 Media, designed to be a highly interactive, premium, and visually stunning digital experience. It showcases our diverse portfolio—from brand reels and ad films to complete visual identities and digital marketing campaigns.

## 🚀 Features

- **Immersive 3D Visuals:** Interactive 3D globes and models powered by Three.js and React Three Fiber.
- **Premium Animations:** Smooth, cinematic page transitions and scroll animations using Framer Motion and GSAP.
- **Buttery Smooth Scrolling:** Integrated with Lenis for a fluid, high-end scrolling experience.
- **Dynamic Portfolios:** Beautifully crafted showcases for branding partners, video productions, and digital marketing case studies.
- **Responsive & Modern Design:** Built with Tailwind CSS, ensuring pixel-perfect layouts across mobile, tablet, and desktop devices.
- **Optimized Performance:** Fast load times utilizing modern web formats like WebP and Vite's highly optimized build pipeline.

## 🛠️ Tech Stack

This project is built with the latest modern web technologies:

* **Framework:** [React 19](https://react.dev/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
* **3D Rendering:** [Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/)
* **Scroll Physics:** [Lenis](https://lenis.studiofreight.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)

## 💻 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have Node.js installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. **Clone the repository** (if applicable)
   ```sh
   git clone https://github.com/your-username/point5-media.git
   ```
2. **Navigate to the project directory**
   ```sh
   cd point5-media
   ```
3. **Install dependencies**
   ```sh
   npm install
   ```
4. **Start the development server**
   ```sh
   npm run dev
   ```
5. **Open your browser**
   Navigate to `http://localhost:3000` to view the website. *(Make sure to leave the terminal running!)*

### Building for Production

To create an optimized production build:
```sh
npm run build
```
To preview the production build locally:
```sh
npm run preview
```

## 📁 Project Structure

```text
├── public/               # Static assets (images, brands, team photos)
├── src/
│   ├── components/       # Reusable UI components (Home, About, Services, UI)
│   ├── data/             # Static content and copy (content.ts)
│   ├── motion/           # Animation utilities and Lenis setup

│   ├── index.css         # Global Tailwind styles & custom CSS
│   ├── App.tsx           # Main application routing and wrapper
│   └── main.tsx          # React application entry point
├── index.html            # HTML template
├── package.json          # Dependencies and scripts
├── tailwind.config.js    # Tailwind configuration
├── tsconfig.json         # TypeScript configuration
└── vite.config.ts        # Vite configuration
```
